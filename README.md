# 🎲 Swift Dice App

Bu uygulama, iOS geliştirme sürecime başlarken Swift programlama dilinin temel yapılarını ve görsellerle etkileşimi öğrenmem amacıyla geliştirdiğim bir uygulama.

## ✨ Özellikler
- **Rastgele Sayı Üretimi:** Butona basıldığında 1 ile 6 arasında rastgele sayılar üretilir.
- **Dinamik UI:** Üretilen sayıya göre zar görselleri anlık olarak güncellenir.
- **Basit Arayüz:** Kullanıcı dostu ve minimal tasarım.

## 🛠️ Teknik Detaylar
- **Dil:** Swift
- **Framework:** UIKit / SwiftUI
